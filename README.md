# iOS-Picterest
- 원티드 프리온보딩 1기 세 번째 과제
- Unsplash API를 활용한 이미지 가져오기 및 가변 세로 길이의 레이아웃 구성
- FileManager, CoreData를 통해 원하는 사진을 선택해 저장하는 기능 구현
